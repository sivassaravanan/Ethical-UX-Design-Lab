# Ethical UX Design Lab

This project explores ethical and unethical practices in UX design.

## Ethical UX Practices

* Clear pricing
* Transparent privacy policies
* Easy navigation
* Honest communication

## Unethical UX Practices

* Hidden fees
* Misleading buttons
* Forced subscriptions
* Difficult account cancellation

## Conclusion

Ethical UX design helps build user trust and provides a better user experience.
